# Component declaration
- ES6
```javascript
class App extends React.Component {
  render() {
    return <h1>Hello from our app</h1>
  }
}
```
- ReactJs api
```javascript
var App = React.createClass({
  render: function() {
    return <h1>Hello from our app</h1>
  }
});
```
